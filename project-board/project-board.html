Features

핵심 기능

Claude Code 개발 흐름을 처음부터 끝까지 안내합니다

📋

6 Phase 워크플로우

Git 초기화 → 기획 → 디자인 → 프론트 → 백엔드 → 마무리, 6단계를 순서대로 진행.

각 Phase는 구체적인 Step과 Claude Code 프롬프트 예시 제공

🎯

다음 할 일 안내

npx claude-code-kit next — 다음 Step과 프롬프트를 즉시 출력. 복사해서 붙여넣기만 하면 됩니다.

매번 "다음에 뭘 해야 하지?" 고민 없이 바로 시작

🤖

CLAUDE.md 자동 주입

Step 완료 시마다 CLAUDE.md에 다음 할 일이 자동 삽입. 새 세션에서도 컨텍스트 유지.

별도로 next를 실행하지 않아도 바로 시작 가능

✅

QC 체크리스트

npx claude-code-kit qc 1 — Phase 완료 전 품질 검토 체크리스트와 Claude Code 프롬프트 자동 생성.

★QA 단계에서 빠뜨리기 쉬운 검토 항목을 자동 안내

🔀

Git 자동화

git-start로 feature 브랜치 생성, git-finish로 커밋 + 푸시 완료.

Phase별 브랜치명·커밋 메시지 컨벤션 자동 적용

📈

진행 상태 추적

npx claude-code-kit status — 전체 진행률과 Phase별 완료 현황을 터미널에 출력.

어디까지 했는지 한눈에 확인 · 웹 대시보드와 자동 연동

How it works

사용 방법

3 steps · 시작부터 배포까지 자동 안내

1

초기화 — npx claude-code-kit init

프로젝트 이름과 기술 스택(Next.js, React+FastAPI 등)을 선택하면 세팅 완료. 첫 번째 할 일이 CLAUDE.md에 자동 주입됩니다.

4가지 기술 스택 템플릿 · 맞춤 프롬프트 자동 생성

2

Claude Code 열기 → 프롬프트 실행

CLAUDE.md를 읽으면 다음 단계와 Claude Code 프롬프트가 이미 준비돼 있습니다. 복사해서 붙여넣고 실행.

"다음에 뭘 해야 하지?" 고민 없이 바로 시작

3

완료 처리 → 자동 업데이트

npx claude-code-kit complete p1s1 실행하면 CLAUDE.md가 다음 단계로 자동 업데이트. Phase 0부터 5까지 반복.

각 Phase 완료 시 QC 체크리스트 + Git 자동화 연계

시작하기

Get started in 1 minute · 1분 안에 시작

① 초기화

# Node.js 18+ 필요
npx claude-code-kit init

# 질문에 답하면 세팅 완료:
프로젝트 이름: My App
기술 스택:
  [1] Next.js + Express
  [2] Next.js 풀스택
  [3] React + FastAPI
  [4] 커스텀

# CLAUDE.md에 첫 할 일 자동 주입 ✓

② 개발 시작

# 다음 할 일 확인
npx claude-code-kit next

# Step 완료 처리
npx claude-code-kit complete p1s1

# 전체 진행 상태 확인
npx claude-code-kit status

# QC 체크리스트 (Phase 완료 전)
npx claude-code-kit qc 1
