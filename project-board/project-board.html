How it works

사용 방법

3 steps to get started · 3단계로 시작

1

Download the files

Download project-board.html and workflow.html and save them anywhere.

두 파일을 다운로드해서 프로젝트 폴더에 저장하세요.

2

Open in browser & start

Double-click to open in browser. Progress is saved automatically to localStorage.

브라우저로 열고 바로 시작. 진행 상태는 자동 저장돼요.

3

Share with your team via GitHub Pages

Put the files in /docs and enable GitHub Pages for team access.

GitHub Pages로 배포하면 팀원 누구나 URL로 접속 가능해요.

GitHub Pages 배포 방법

Deploy in 2 minutes · 2분 안에 팀과 공유

① 파일 추가 & Push

# 저장소 클론
git clone https://github.com/
  astudyinglady-max/
  clauce-commin-kit.git
cd clauce-commin-kit

# docs 폴더에 파일 저장
mkdir docs
# index.html     ← 랜딩 페이지
# project-board.html ← 프로젝트 보드
# workflow.html  ← 작업 순서 v5

git add docs/
git commit -m "docs: add toolkit"
git push origin main

② GitHub Pages 설정

# GitHub 저장소에서:
1. Settings 클릭
2. Pages 메뉴 이동
3. Source:
   Deploy from branch
4. Branch: main
   Folder: /docs
5. Save 클릭

# 배포 완료 후 접속:
https://astudyinglady-max
  .github.io/clauce-commin-kit

# 각 파일 직접 접속:
/project-board.html
/workflow.html
