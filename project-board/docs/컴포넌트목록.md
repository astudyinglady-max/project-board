## 1. 버튼 (Button)

| 컴포넌트 | 클래스 | 사용 페이지 | 상태 | 설명 |
|----------|--------|------------|------|------|
| Primary 버튼 | `.btn-primary` | 랜딩, 보드, 워크플로우 | default / hover / focus / disabled | 주요 액션(저장, 생성, CTA). hover 시 배경 진하게, 약간 위로 이동 |
| Outline 버튼 | `.btn-outline` | 랜딩, 보드 | default / hover / focus / disabled | 보조 액션. hover 시 테두리·텍스트 Primary 색 |
| Green 버튼 | `.btn-green` | 랜딩 | default / hover | 성공/시작 액션 |
| Danger 버튼 | `.btn-danger` | 보드 | default / hover / focus / disabled | 삭제 액션. 빨간 배경·텍스트 |
| Text 버튼 | `.btn-text` | 워크플로우 | default / hover | 최소 강조(초기화, 파일 저장 등) |
| 복사 버튼 | `.copy-btn` | 랜딩, 워크플로우 | default / done(복사됨) | 프롬프트·코드 복사. 클릭 후 "✓ 복사됨"으로 변경 후 1.5초 뒤 원복 |
| Reset 버튼 | `.reset-btn` | 워크플로우 | default / hover | 전체 초기화, 파일로 저장 |

**사용 예시**

- 보드: "프로젝트 추가", "내보내기", "보드 열기", "삭제"
- 워크플로우: "복사", "전체 초기화", "파일로 저장"
- 랜딩: "다운로드", "체험하기", "GitHub", 설치 코드 블록의 "복사"

## 2. 입력 (Input / Select / Textarea)

| 컴포넌트 | 클래스/요소 | 사용 페이지 | 상태 | 설명 |
|----------|-------------|------------|------|------|
| 텍스트 입력 | `input[type="text"]`, `#inp-name` 등 | 보드 | default / focus / placeholder / error | 프로젝트명, 설명 등 |
| 날짜 입력 | `input[type="date"]` | 보드 | default / focus | 시작일, 마감일 |
| Select | `select` | 보드 | default / focus / open | 상태(진행중/완료 등) |
| Textarea | `textarea`, `#inp-desc` | 보드 | default / focus / placeholder | 프로젝트 설명 |

**상태 규칙**

- **focus**: outline 없음, `border-color: var(--indigo)`, `box-shadow` 링
- **error**: `border-color: #dc2626`, `background: #fff5f5`
- **disabled**: `opacity: 0.5`, `cursor: not-allowed`

**사용 예시**

- 보드: 새 프로젝트 추가 폼(프로젝트명*, 상태, 시작일, 마감일, 설명)

## 3. 카드 (Card)

| 컴포넌트 | 클래스 | 사용 페이지 | 상태 | 설명 |
|----------|--------|------------|------|------|
| 프로젝트 카드 | `.project-card` | 보드 | default / hover / selected | 프로젝트 목록 항목. 클릭 시 선택, "보드 열기"·"삭제" 버튼 |
| 폼 카드 | `.form-card` | 보드 | default | 새 프로젝트 추가 폼 래퍼 |
| 도구 카드 | `.tool-card` | 랜딩 | default / hover | workflow / board / landing 소개 카드 |
| 피쳐 카드 | `.feat-card` | 랜딩 | default / hover | 6개 기능 소개 |
| Phase 블록 | `.phase-block`, `.phase` | 보드, 워크플로우 | default / open(펼침) | Phase 단위 블록. 좌측 컬러 바, 클릭 시 본문 펼침/접힘 |
| Step 항목 | `.step-item`, `.step` | 보드, 워크플로우 | default / open / qa-step | Step 단위. 체크/펼침, ★QA 스텝은 뱃지 강조 |

**카드 공통**

- **hover**(인터랙티브 카드): 테두리 Primary, 그림자, `translateY(-2px)`(선택)
- **selected**: 프로젝트 카드만. 선택된 프로젝트 강조

## 9. 모달 (Modal)

| 컴포넌트 | 클래스 | 사용 페이지 | 상태 | 설명 |
|----------|--------|------------|------|------|
