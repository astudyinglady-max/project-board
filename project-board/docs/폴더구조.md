## 폴더 역할 요약

| 폴더/파일 | 역할 | 수정 권한 |
|-----------|------|----------|
| `docs/` | GitHub Pages로 서빙되는 HTML + 기획 문서 | 자유롭게 수정 |
| `kit/` | 로컬 CLI (프로젝트 맞춤 수정 가능) | 자유롭게 수정 |
| `.kit/` | 업스트림 git submodule | **수정 금지** |
| `.workflow/config.json` | 워크플로우 단계 정의 | 수동 편집 가능 |
| `.workflow/state.json` | 현재 진행 상태 | CLI 명령어로만 수정 |
| `.workflow/state.js` | HTML용 상태 브릿지 | **자동 생성, 수정 금지** |
| `.workflow/status.md` | 사람이 읽는 상태 요약 | **자동 생성, 수정 금지** |
| `.workflow/templates/` | Step별 문서 초안 템플릿 | 자유롭게 수정 |
| `.github/` | GitHub 이슈·PR 템플릿 | 자유롭게 수정 |
| `agents/` | Claude Code 에이전트 정의 | 자유롭게 수정 |
| `skills/` | Claude Code 슬래시 커맨드 | 자유롭게 수정 |
| `sprint/` | 스프린트별 로그 및 이슈 | 자유롭게 수정 |
