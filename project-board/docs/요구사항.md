## 제외 범위 (Out of Scope)

이번 스프린트에서 의도적으로 제외하는 기능들입니다.

- **서버 사이드 로직 (DB, API)** — 이유: 서버리스 정적 배포 원칙, GitHub Pages 타겟
- **사용자 로그인/회원가입** — 이유: 개인 사용 툴킷으로 인증 불필요
- **실시간 다중 사용자 동시 편집** — 이유: 협업 기능은 백엔드 없이 구현 불가
- **모바일 네이티브 앱 (iOS/Android)** — 이유: 브라우저 전용 툴킷 범위 초과
- **JWT 인증 / HTTPS 강제** — 이유: 로컬 파일 또는 GitHub Pages 정적 서빙 환경

## 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 성능 | 첫 로드 2초 이내 (정적 파일, CDN 없음) |
| 접근성 | WCAG AA 기준 (4.5:1 컬러 대비 충족) |
| 반응형 | 360px ~ 1440px 전 구간 사용 가능 |
| 의존성 | 제로 — 외부 라이브러리, 번들러, 빌드 도구 없음 |
| 배포 | GitHub Pages 정적 배포 (서버 불필요) |
| 브라우저 지원 | Chrome, Safari, Firefox 최신 2버전 |
| 데이터 저장 | localStorage (서버 없이 브라우저 내 영속성) |
